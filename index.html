Dynasty is a promise-based Amazon DynamoDB client for Node.

DynamoDB is Amazon's high performance, all SSD backed, fully managed NoSQL database offering. This library was built in an attempt to make it a bit more easy to work with.

This library is a bit in its early stages, so if you're looking for things to be 100% stable and feature complete, it may be worth looking at other options unless you're somewhat adventurous. And if you're the adventurous type, please help out and send some pull requests!

Dynasty is open source software and is released under the developer and business-friendly MIT license.
